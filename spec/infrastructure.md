# Moodo インフラ設計書

## 方針

本アプリはローカル起動のみを対象とし、クラウドへのデプロイ・外部サービスへの依存は一切ない。
そのため、従来の意味でのインフラ設計（サーバー構成、クラウドリソース、CI/CD、ネットワーク設計など）は存在しない。

## ローカル実行環境

| 項目 | 内容 |
|---|---|
| 実行環境 | 開発者のローカルマシン（Windows想定） |
| フロントエンド起動 | `npm run dev`（Node.js） |
| バックエンド起動 | `uvicorn main:app --reload`（Python + uv） |
| データ | `tasks.json`（ローカルファイル） |
| 外部サービス依存 | なし（AI評価はClaude.aiへの手動貼り付けで運用） |

## 将来の検討事項

現時点では対象外だが、将来的に以下を検討する可能性がある。

- クラウドデプロイ（複数デバイスからのアクセスやデータ同期が必要になった場合）
- データベースへの移行（タスク数が増えてJSONファイルの管理が煩雑になった場合）
